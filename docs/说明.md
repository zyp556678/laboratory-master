# 张宇鹏的实验室管理系统完整项目结构与功能说明

## 一、项目概述

张宇鹏的实验室管理系统是基于SpringBoot开发的实验室管理软件，采用前后端分离架构。该系统旨在提高实验室的效率、安全性和管理水平，简化日常操作，并提供实验室数据的准确记录和分析。

## 三、核心功能模块详细说明

### 1. 用户管理模块

#### 相关文件及功能说明：

**实体类：[User.java](file:///D:/Desktop/Study/laboratory-master/laboratory-server/src/main/java/com/laboratory/model/entity/User.java)**
- 包含账号、昵称、密码、性别、头像、邮箱、状态、备注和角色ID等字段
- 继承自BaseEntity，包含基本的创建时间和更新时间字段

**控制器：[UserController.java](file:///D:/Desktop/Study/laboratory-master/laboratory-server/src/main/java/com/laboratory/controller/UserController.java)**
- `/admin/user/page/{pIndex}/{pSize}` POST - 分页查询所有用户信息
- `/admin/user/profile` PUT - 修改个人信息
- `/admin/user/updPwd` PUT - 修改密码
- `/admin/user/resetPwd/{id}` GET - 重置密码

**服务接口：[UserService.java](file:///D:/Desktop/Study/laboratory-master/laboratory-server/src/main/java/com/laboratory/service/UserService.java)**
- login - 用户登录认证
- getAll - 分页获取所有用户
- getInfo - 根据token获取用户信息
- updatePwd - 更新用户密码

**服务实现：[UserServiceImpl.java](file:///D:/Desktop/Study/laboratory-master/laboratory-server/src/main/java/com/laboratory/service/impl/UserServiceImpl.java)**
- 实现登录逻辑，包括密码验证、JWT token生成
- 实现密码加密存储（MD5）
- 实现用户信息查询和更新
- 实现管理员和当前用户不能被删除的安全机制

**Mapper接口：[UserMapper.java](file:///D:/Desktop/Study/laboratory-master/laboratory-server/src/main/java/com/laboratory/mapper/UserMapper.java)**
- 继承MyBatis-Plus的BaseMapper，提供基本的CRUD操作

**前端API：[sys/user.js](file:///D:/Desktop/Study/laboratory-master/laboratory-admin/src/api/sys/user.js)**
- 封装用户相关的API调用

**前端页面：[sys/user/](file:///D:/Desktop/Study/laboratory-master/laboratory-admin/src/views/sys/user/)**
- 用户管理界面，包括用户列表展示、新增、编辑、删除等功能

#### 功能特性：
- 用户查看、新增、修改、删除
- 用户密码重置
- 头像上传
- 权限角色分配（管理员、教师、学生）

## 四、角色权限体系

系统包含三种角色，各自具有不同的权限：

1. **管理员**：拥有系统的全部权限，可进行所有管理操作
2. **教师**：可以管理自己负责的课程、项目及相关实验室预约
3. **学生**：可以查看相关信息，提交预约申请和设备报修

## 五、技术栈

### 前端技术栈
- Vue：前端核心框架
- VueX：状态管理
- Vue Router：路由控制
- Axios：HTTP请求
- Element-UI：UI组件库
- ECharts：可视化图表
- Vite：构建工具

### 后端技术栈
- SpringBoot：快速构建微服务框架
- MyBatisPlus：ORM框架
- PageHelper：分页插件
- JWT：JSON Web Token认证
- EasyExcel：导入导出框架
- Swagger2：接口文档
- MySQL：主数据库
- Minio：文件存储

## 六、运行环境要求

- MySQL 5.7+
- JDK 1.8+
- Maven 3.6+
- NodeJS 8.9+
